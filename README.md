# Profile Card

A clean, responsive profile card built with HTML and CSS — a practice project for honing front-end layout and styling skills.

## Features

- Centered card layout with rounded corners and box shadow
- Profile image, name, tagline, follower/comment stats, and a follow button
- Font Awesome icons for clean UI accents
- Responsive design using flexbox

## Preview

Open `Profile-Card.html` in any browser to view the card.
